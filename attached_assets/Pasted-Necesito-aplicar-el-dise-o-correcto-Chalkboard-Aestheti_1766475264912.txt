Necesito aplicar el diseño correcto "Chalkboard Aesthetic" a tcf-webapp.

PASO 1: COPIAR ASSETS desde tcf-brand-master

Primero, necesito que crees la carpeta public/ si no existe, y luego copies estos assets desde el repo tcf-brand-master:

Clona el repo tcf-brand-master:
git clone https://github.com/RUPERDFN/tcf-brand-master.git temp-brand

Luego copia los assets necesarios:
mkdir -p public/fonts
mkdir -p public/brand
mkdir -p public/icons

cp temp-brand/fuentes/Cheveuxdange.ttf public/fonts/
cp temp-brand/fondos/bg.png public/brand/bg-chalkboard.png
cp temp-brand/fondos/textura-grunge-oscura.jpg public/brand/bg-texture.jpg
cp temp-brand/logos/logo.PNG public/brand/logo.png
cp temp-brand/logos/tomate.png public/icons/
cp temp-brand/logos/zanahoria.png public/icons/
cp temp-brand/logos/pimiento.png public/icons/
cp temp-brand/logos/limon.png public/icons/

# Limpiar
rm -rf temp-brand
